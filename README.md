# エンタメ作品レビュー管理Webアプリ

## 公開URL

https://aki-suya.github.io/entame-review-app/

## 概要

映画、アニメ、漫画、ゲーム、音楽などのエンタメ作品のレビューを管理できるWebアプリです。

作品名、ジャンル、評価、感想を登録し、一覧表示・編集・削除・検索・評価順の並び替えができます。

## 作成した目的

就職活動やインターン応募で、Webアプリを自分で制作した経験として説明できる制作物を作ることを目的にしました。

また、エンタメ業界に関心があるため、自分の興味に近い題材として、視聴・プレイした作品を整理できるアプリを制作しました。

この作品では、HTML・CSS・JavaScriptを使った基本的なWebアプリ制作の流れを学ぶことを目指しました。

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages

## 主な機能

- 作品名の登録
- ジャンルの選択
- 評価の登録
- 感想の登録
- レビュー一覧表示
- レビューの編集
- レビューの削除
- キーワード検索
- 評価順の並び替え
- localStorageによるデータ保存

## 工夫したこと

- 映画、アニメ、漫画、ゲーム、音楽など、複数のエンタメ作品を管理できるようにジャンルを選択式にしました。
- 登録したレビューを見返しやすいように、カード形式で表示しました。
- キーワード検索機能を追加し、登録した作品を探しやすくしました。
- 評価順の並び替え機能を追加し、高評価の作品を確認しやすくしました。
- localStorageを使い、ページを更新してもレビュー情報が残るようにしました。

## 学んだこと

HTMLで画面の構造を作り、CSSで見た目を整え、JavaScriptで動きをつける基本的な流れを学びました。

また、ユーザーが入力した情報をオブジェクトとして扱い、配列で管理する方法を学びました。

さらに、DOM操作によって画面の内容を動的に更新する方法や、localStorageを使ってブラウザ上にデータを保存する方法を理解しました。

## 反省・今後の改善点

初めてのWebアプリ制作だったため、最初はHTML、CSS、JavaScriptの役割を理解するのに時間がかかりました。

特に、入力されたデータを配列で管理し、画面に再表示する処理の流れを理解するのが難しかったです。

今後は、ジャンル別の絞り込み、作品画像の登録、より見やすいデザインへの改善、スマートフォンでの操作性向上などに取り組みたいです。
