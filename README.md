# エンタメ作品レビュー管理Webアプリ

## 公開URL

https://新しいユーザー名.github.io/entame-review-app/

## 概要

映画、アニメ、漫画、ゲーム、音楽などのエンタメ作品のレビューを管理できるWebアプリです。  
作品名、ジャンル、評価、感想を登録し、一覧表示・削除・検索ができます。

プログラミング初心者として、HTML・CSS・JavaScriptの基礎を学びながら制作しました。

## 作成した目的

就職活動やインターン応募で「自分でWebアプリを制作した経験」として説明できる制作物を作ることを目的にしました。

単に画面を作るだけでなく、JavaScriptを使って登録・削除・検索・データ保存などの機能を実装しました。

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

## 主な機能

- 作品名の登録
- ジャンルの選択
- 評価の登録
- 感想の登録
- レビュー一覧表示
- レビュー削除
- 作品名・ジャンル・感想による検索
- localStorageによるデータ保存

## 工夫した点

- カード型レイアウトにして、登録したレビューを見やすく表示した
- 評価を星で表示し、直感的に分かりやすくした
- localStorageを使い、ページを更新してもレビューが残るようにした
- 検索機能を追加し、登録数が増えても目的のレビューを探しやすくした
- スマートフォンでも見やすいように、レスポンシブ対応を行った

## 苦労した点

JavaScriptでHTMLの要素を取得し、ボタンを押したときに画面を書き換える処理の理解に苦労しました。  
特に、レビューを配列として管理し、localStorageに保存してから再表示する流れを理解するのが難しかったです。

しかし、機能を小さく分けて実装することで、登録・削除・検索・保存の仕組みを少しずつ理解できました。

## 今後追加したい機能

- 評価順で並び替える機能
- ジャンルごとの絞り込み機能
- 編集機能
- 登録日を表示する機能
- Firebaseなどを使った本格的なデータ保存

## 制作を通して学んだこと

HTMLで画面構造を作り、CSSで見た目を整え、JavaScriptで動きをつけるというWebアプリ制作の基本的な流れを学びました。  
また、ユーザーが入力したデータを配列やオブジェクトで管理し、localStorageに保存することで、簡易的なデータ管理機能を実装できることを学びました。
